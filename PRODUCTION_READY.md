## 🚀 خطوات البدء

### 1. تسجيل الدخول
```
https://langford-system.vercel.app/login
```

### 2. إنشاء المستخدمين
1. اذهب إلى: Settings → Users → Add New User
2. أنشئ:
   - **Coordinator** (منسق إداري) - يربط الطلاب بالدورات والمدرسين
   - **Sales** (مبيعات) - يضيف الطلاب ويسجل المدفوعات
   - **Instructor** (مدرس) - يسجل الحضور فقط

### 3. إضافة الطلاب
1. Students → Add Student
2. املأ البيانات الأساسية
3. اختر الدورة المهتم بها (اختياري)

### 4. تسجيل الطلاب في الدورات
1. افتح صفحة الطالب
2. اذهب إلى تبويب Enrollments
3. اضغط "Enroll in Course"
4. اختر الدورة والتاريخ والرسوم

### 5. تسجيل المدفوعات
1. افتح صفحة الطالب
2. اذهب إلى تبويب Payments
3. اضغط "Record Payment"
4. املأ البيانات واختر الدورة

### 6. تعيين المدرسين للدورات
1. Settings → Courses
2. اضغط Edit على أي دورة
3. اختر Instructor من القائمة

### 7. تسجيل الحضور (للمدرسين)
1. سجل دخول كمدرس
2. اذهب إلى Attendance
3. اختر الدورة والتاريخ
4. سجل حضور الطلاب

```bash

```

```bash

```

```bash

```

```bash
npx firebase deploy --only firestore:rules
```
